# Green Habit Tracker App

GreenSteps is a mobile application that encourages users to adopt sustainable habits aligned with the Singapore Green Plan 2030. Users can add sustainable habits and record habit completions to earn points. The application uses a web service and database to store habit data and calculate total points earned.

## Installation


To set up and run the app:

1. Make sure you have Node.js and npm installed.
2. Install Expo CLI globally (if not already installed):
	```
	npm install -g expo-cli
	```
3. Install project dependencies:
	```
	npm install
	```
4. Start the development server:
	```
	npm start
	```
5. Use the Expo Go app on your mobile device or an emulator to preview the app.

## Main Packages Used

- **expo**: For building and running the React Native app
- **react**: Core library for building user interfaces
- **react-native**: Framework for building native apps using React
- **expo-status-bar**: For status bar management
- **@react-navigation/native** and **@react-navigation/native-stack**: For navigation between screens
